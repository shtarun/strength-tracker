# User Flows Documentation

This folder contains visual documentation of all major user flows in StrengthTracker, with a focus on AI-powered features.

## Contents

- [AI Workout Plan Generation](./ai-plan-generation.md) - Flow for creating workout plans using AI
- [AI Workout Customization](./ai-workout-customization.md) - Flow for AI-powered workout adjustments based on readiness
- [AI Exercise Substitution](./ai-exercise-substitution.md) - Flow for getting AI exercise alternatives
- [Plan Progression](./plan-progression.md) - Flow for plan week advancement and completion
- [Workout Execution](./workout-execution.md) - Complete workout flow from start to finish

## Mermaid Charts

All flow diagrams use [Mermaid](https://mermaid.js.org/) syntax which renders natively on GitHub.

To view these charts:
1. Open the markdown files on GitHub
2. The mermaid diagrams will render automatically
3. Or use a Mermaid-compatible markdown viewer
